# Harnessing Large Language Models for Radiomics Analysis in Precision Oncology

## Project Overview

This project investigates whether large language models can provide meaningful representations for radiomics-based cancer classification tasks.
Using 24 public radiomics datasets, we compared traditional machine learning approaches with LLM-based representations generated from BERT and LLaMA models.
The study aims to evaluate the strengths and limitations of applying LLMs to structured radiomics data.
